# pyano
Humanize the MIDI sound, then we could input it to DAW and create realistic piano sound

## About
Features:
- Randomize note velocity with range
  - lower than maximum
  - higher than minimum
- Add pedal to MIDI sound
- Simplify the MIDI sound
